# Exercício 14

Crie uma função que receba uma string no formato **hh:mm:ss** e retorne um objeto que tenha as propriedades **hours**, **minutes** e **seconds**. Os valores da propriedades devem ser **números**, não strings.

## Exemplo

```jsx
Entrada

"01:21:53"

Saída

{
    hours: 1,
    minutes: 21,
    seconds: 53
}
```
