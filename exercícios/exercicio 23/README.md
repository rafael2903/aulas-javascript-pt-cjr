# Exercício 23

Para esse exercício, utilize a variável *stock* presente no arquivo *data.js*. Crie uma função que receba como parâmetro o nome de uma comida e retorne um booleano indicando se a comida está em estoque. Utilize o método [includes](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Reference/Global_Objects/Array/includes).
